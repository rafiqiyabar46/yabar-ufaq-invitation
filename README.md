# Ufaq & Yabar Wedding Invitation

A static, responsive wedding invitation designed for GitHub Pages.

## Deploy

Push this directory to a GitHub repository, then enable **Settings → Pages → Deploy from a branch**. No build step is required.

## Local use

Open `index.html` in a browser. Service-worker caching requires a local server or GitHub Pages because browsers only allow it on secure origins.

## Notes

The invitation uses Google Fonts plus optional GSAP, ScrollTrigger, and Lenis CDNs. Content and core layout stay functional if a CDN is unavailable.
